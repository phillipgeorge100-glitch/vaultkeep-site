Frequently asked questions

Where is my data stored? +
On your device. VaultKeep stores all portfolio data in your browser's local storage or, for the desktop app, in an encrypted file on your machine. Nothing is transmitted to any server.
Do I need an internet connection? +
No. VaultKeep works fully offline. Price feeds and the AI assistant require internet when you choose to use them, but the core application — your dashboard, calculations, and portfolio — works without any connection.
How does the AI assistant work without sending my data to VaultKeep? +
You provide your own API key from providers like OpenAI or Anthropic. When you ask a question, VaultKeep sends your query directly from your browser to that provider. We are never in the middle, never see the conversation, and never store it.
What happens if I lose my device? +
VaultKeep includes JSON export for full backups. We recommend exporting regularly. The desktop app stores data locally, so standard device backup practices apply.
Is VaultKeep regulated by the FCA? +
VaultKeep is a portfolio tracking tool, not a financial service. It does not provide investment advice, manage client funds, or execute trades. All calculations are informational. You make your own decisions.
Can I use VaultKeep on multiple devices? +
The web version works in any modern browser. The desktop app runs on Windows and macOS. Data is stored locally per device. You can transfer data between devices using the JSON export and import feature.
